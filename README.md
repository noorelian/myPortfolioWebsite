# myPortfolioWebsite
This is my personal portfolio website, built using HTML and CSS, to showcase my projects, experiences, and skills in a clean, organized way. The website is divided into multiple sections that highlight my background, technical abilities, and achievements.

Features

Responsive Design: Works on desktop, tablet, and mobile devices.

Sectioned Layout: Includes sections such as:

Home / About Me – Introduction and background information

Skills – Technical and soft skills overview

Projects – Highlighted personal and academic projects

Experience / Activities – Work experience, volunteering, and leadership roles

Contact – Easy way to get in touch

Technologies Used

HTML5 – Structure and content

CSS3 – Styling and layout
